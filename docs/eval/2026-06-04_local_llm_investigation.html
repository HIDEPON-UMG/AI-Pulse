ローカル LLM 抽出置換 調査レポート

AI-Pulse の記事抽出を Gemini API からローカル LLM（Ollama / RTX5080 16GB）へ置換できるか — 4 段階の eval と盲検ジャッジ採点による定量判断。

N=10 盲検採点 3 contestant gemini-flash judge 結論: 35B≈flash-lite 互角

エグゼクティブサマリ

品質は 35B-A3B ≈ flash-lite の互角（強みの交換）。qwen3:14b は明確に 3 番手。盲検ジャッジ総合は 35B-A3B 3.80 > flash-lite 3.38 > qwen3:14b 3.02（各 1-5）。ただし総合の逆転は rationale 次元がほぼ単独で駆動しており、factual（事実忠実度）はむしろ flash-lite が上。

動機の正確な理解

ローカル化は「品質向上」ではない。コストは flash-lite で既に $0.015/76件。動機は APIクォータ非依存・オフライン自走・考察記事(essay)の同モデル一本化・環境活用。品質は「維持」が正しい理解。

Claude 推奨：抽出は 35B-A3B 一本化

3.80盲検総合（最良）
~22分76件直列（夜間OK）
×0API クォータ依存

essay と同モデル・判定品質最良・運用単純。qwen3:14b は見送り（品質3番手・速度差は夜間バッチで僅少）。ハイブリッド配線（local 失敗/GPU 占有時に Gemini フォールバック）。最大の watch-item は factual 忠実度（ローカルは固有名詞捏造がやや多い）。

盲検ジャッジ採点（最も厳密な比較）

方法

N=11（本文取得成功）/ 採点 10 件（3 モデル全成功サンプル・1 件はジャッジ 503 で欠測）。評価条件を計画アーキに合わせて公平化：強調契約を無効化（強調はコード付与へ移す決定済）+ maxLength を 400 に緩和（語中切れ除去）。3 モデル同条件・temp 0.4。gemini-2.5-flash が本文を見て盲検採点（候補 A/B/C を index ローテーションで shuffle・モデル名隠蔽・マークアップは評価対象外）。

バイアス注記

ジャッジが Gemini 系のため flash-lite 有利の系統誤差リスクあり → rubric を客観軸（事実=本文照合 / 完全性=切れ / rationale=ラベル反復減点）に寄せて緩和。結果はその予想に反し 35B が flash-lite を上回ったため、35B 評価はバイアスを差し引いても堅い。

全モデル共通

⑤ 一部の捏造は本文 3000 字打切り由来。devin 系で全モデルが「89% のコードを Devin がコミット」や過去調達額を捏造（factual 1-2）。モデル差でなく MAX_BODY_CHARS=3000 の打切りで本文が欠け、モデルが学習知識で穴埋めした結果。本文取得改善で全モデル改善余地。

「強みの交換」の構図（最重要の解釈）

次元勝者差の内容

summary35B ≈ flash-liteほぼ同等

結論

総合では 35B が上だが、それは 35B が rationale で大きく勝ち、flash-lite が factual で勝つという交換。コア抽出（factual/summary/points）は flash-lite ≳ 35B。つまり品質は実質互角。ローカル化は「品質向上」でなく「品質維持 + 非機能要件（クォータ/offline/essay 一本化）」で評価すべき。

35B-A3B

解決済みの重要発見

flash-lite の rationale 低スコアは実在でした。本番 events.jsonl は rationale 充実ですが、これは flash（gemini-2.5-flash）出力で確定済み。eval したのは flash-lite。つまり flash=充実 / flash-lite=不安定にラベル退化（7/10）/ 35B=一貫充実。

→ 副次発見：以前結論づけた flash→flash-lite のコスト切替は rationale 品質を劣化させるリスク（4サンプルでは偶然 full だった）。35B なら flash 級 rationale をクォータ非依存で得られるため、この点は 35B 推奨を補強。

未解決の論点 / 留意

  • factual 捏造の本文打切り寄与：MAX_BODY_CHARS 引上げ + 本文抽出改善で全モデルの factual が上がる可能性。ローカル採否と分離して検討。
  • N=10・単一 Gemini 系ジャッジの限界。サンプル増 / 別系統ジャッジ（Claude 等）で確度は上がるが、方向性（35B≈flash-lite・qwen3 劣後）は安定。
